## 用 yarn 安装 vue-devtools

```
git clone https://github.com/vuejs/vue-devtools.git
git clone https://gitee.com/mirrors/vue-devtools.git 【国内】
yarn install
yarn run build
```

## chrome 扩展程序

- chrome://extensions/
- 加载已解压的扩展程序
- vue-devtools\packages\shell-chrome

## 用 vite 创建 vue 项目

```
yarn create vite
yarn install
yarn run dev
```
